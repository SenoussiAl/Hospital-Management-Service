# User Sub-Domain

- **Category:** Generic

- **Description:** This subdomain is responsible for managing users and their roles within the application. It captures the user's personal information, role, and other relevant information. It's responsible for authenticating users and authorizing them to access certain resources.

- **Rationale:** Creating and managing users is a reusable solution used across the system. This domain mainly supports use cases: Create User, Update User, Delete User, login, logout etc...
